# INSTALACIÓN
1. Clonar el repositorio en su equipo.
```Shell
git clone https://github.com/marcelonj/pfo1_programacion_sobre_redes 
```

2. Iniciar el servidor.
```Shell
py servidor.py
```
> [!NOTE]
> Para finalizar el servidor presione `CTRL+C en la consola en la que se inició el mismo.

2. Iniciar el cliente.
```Shell
py cliente.py
```

> [!NOTE]
> Para finalizar el cliente envíe la palabra "éxito" como mensaje.
